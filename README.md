## 🚀 Featured Project

### [CBC Disease Prediction API](https://github.com/Jijnash2636/CBC-Disease-Prediction)

An AI-powered clinical decision support system that analyzes Complete Blood Count (CBC) test results to predict potential diseases with high accuracy.

- **95%+ Accuracy** on disease classification
- **FastAPI** backend with async processing
- **Real-time inference** for clinical diagnostics
- **Explainable AI** with SHAP-based interpretations

## ⚡ Quick Start

```bash
# Clone the repository
git clone https://github.com/Jijnash2636/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Open `http://localhost:8080` to view the portfolio.
